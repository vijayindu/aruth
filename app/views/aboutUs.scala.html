About Project Aruth
Problem Statement
The ambiguity of a word leads to confusion, especially for computers. Sinhala has a wide range of words which have multiple meanings in different contexts. These are called polysemous words. These polysemous words cause problems to tasks such as translating from Sinhala to another language and summarizing texts. A tool is needed to identify the correct sense of a polysemous word in a given context.
Ex: ඇයට කෑම පිසින්න පුලුවන්
The word “පිසින්න” has several senses such as “cook” and “wipe”.Therefore it can be translated to “She can cook food” or “She can wipe food”. The problem is to correctly identify “cook” as the sense meant in the context.
Objectives
To identify the Word Sense Disambiguation technique most suitable for Sinhala and to create a tool using that technique to identify the correct sense of an ambiguous Sinhala word in a given context with high accuracy.
Project Overview
Word Sense Disambiguation (WSD) is an open problem in natural language processing and ontology, which governs the process of identifying which sense of a polysemous word (i.e. meaning) is used in a sentence. Number of researches have been carried throughout decades and now English has a fairly good WSD methodologies. Some of them are dictionary based methods that use the knowledge about the words encoded in those lexical resources. Corpus-based WSD methods are another area of research.
At the moment, Sinhala language doesn’t have a tool for WSD. But there is a Sinhala WordNet which is a lexical dataset for Sinhala language. It groups Sinhala words into synonyms and provide definitions and has several semantic relations between those synonyms set. Sinhala WordNet, a machine readable dictionary, can be used as the data source to a Sinhala WSD tool.
WSD Techniques Identified So Far
WSD algorithms can be classified under rule-based algorithms and machine learning algorithms. Some of the WSD techniques published by various researchers are listed down below.

Rule-based Algorithms

  • Lesk Algorithm
  • Simplified Lesk Algorithm
  • Selectional Preference
  • Semantic Similarity (Contextual Semantic Relatedness)
  • Lexical Chains
  • Heuristic based approaches


Machine Learning Algorithms

  • Naive Bayesian Classifier
  • Neural Networks [Leacock]
  • Decision Tree/List Learners
Importance of a WSD Techniques for Sinhala Language
Proper word sense disambiguation for Sinhala is important for many NLP applications and it opens paths to ,

  • Accurate translation from Sinhala to other languages
  • Create proper search methodologies for internet surfing for Sinhala resources
  • Text summarization
  • Better information retrieval systems for Sinhala
  • Word processing and spell checking
  • Content analysis
  • Information extraction
  • Text to Speech translation
